# Clustering_Moviles

Se analizó una base de datos de dispositivos móviles, realizando aprendizaje no supervisado para encontrar grupos de interés. Se realizó un primer clustering para separar tablets de teléfonos móviles, se hizo selección de características y se realizó un segundo clustering para separar teléfonos móviles en gama baja, media y alta. Finalmente, se describen brevemente algunos patrones en los datos que pueden explotarse para toma de decisiones en marketing.
